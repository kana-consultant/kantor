Activity Tracker Settings

Connection

Cara termudah untuk user biasa: buka dashboard web KANTOR, masuk ke menu Operational -> Activity Tracker, download paket extension di sana, pasang sekali di Chrome, lalu klik Hubungkan & Aktifkan.

Form di bawah disediakan sebagai fallback manual. Gunakan API URL http://localhost:3000/api/v1 untuk development. Access token bisa diambil dari DevTools web KANTOR pada key local storage kantor-auth.
